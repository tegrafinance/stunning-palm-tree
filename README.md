# Stunning Palm Tree

An Automated Multi-Agent Web3 Trading & Risk Assessment Pipeline that connects technical analysis signals with real-time on-chain data validation before execution.

## Features
- **Ingestion & Routing Agent:** Monitors incoming trading alerts and fetches real-time on-chain data metrics.
- **Reasoning & Risk Agent:** Evaluates combined data using multi-step chain-of-thought prompting to calculate optimal execution vectors.
- **Execution Agent:** Dispatches payloads securely to execute automated smart contract interactions.

## Getting Started

### Prerequisites
- Node.js (v16+)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/tegrafinance/stunning-palm-tree.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Agent
To start the development server:

```bash
npm start
```

The API will be available at `http://localhost:3000`.
